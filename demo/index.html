vaadin-combo-box Basic

Opening and closing

vaadin-combo-box dropdown list can be opened by cliking/tapping the element or programmatically with the open() method

Clicking outside the dropdown or tapping the close icon on fullscreen mode closes it. vaadin-combo-box also has an explicit close() method.

var combobox = combobox || document.querySelector("vaadin-combo-box"); combobox.items = ['foo', 'bar', 'baz']; var togglebutton = document.querySelector("#togglebutton"); togglebutton.addEventListener('click', function() { if (combobox.opened) { combobox.close(); } else { combobox.open(); } });

Selecting a value

Selecting a value from the dropdown list or typing a value to the textfield changes vaadin-combo-box's value property and fires a value-changed event.

Changing the value property explicitly also updates the visible fields.

var combobox = combobox || document.querySelector("vaadin-combo-box"); combobox.items = ['foo', 'bar', 'baz']; combobox.value = 'bar'; combobox.addEventListener('value-changed', function() { console.log("New value: " + combobox.value); });
