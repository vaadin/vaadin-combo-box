vaadin-combo-box Basic

Opening and closing

vaadin-combo-box dropdown list can be opened by cliking/tapping the element or programmatically with the open() method

Clicking outside the dropdown or tapping the close icon on fullscreen mode closes it. vaadin-combo-box also has an explicit close() method.

var combobox = combobox || document.querySelector("vaadin-combo-box"); combobox.items = ['foo', 'bar', 'baz']; var togglebutton = document.querySelector("#togglebutton"); togglebutton.addEventListener('click', function() { if (combobox.opened) { combobox.close(); } else { combobox.open(); } });

Selecting a value

Selecting a value from the dropdown list changes vaadin-combo-box's value property and fires a value-changed event.

Changing the value property explicitly also updates the visible fields.

var combobox = combobox || document.querySelector("vaadin-combo-box"); combobox.items = ['foo', 'bar', 'baz']; combobox.value = 'bar'; combobox.addEventListener('value-changed', function() { console.log("New value: " + combobox.value); });

Using as a form field

vaadin-combo-box is extended with IronFormElementBehavior so it can be used as a field in an iron-form.

var form = form || document.querySelector('form'); var combobox = form.querySelector('vaadin-combo-box') combobox.items = ['foo', 'bar', 'baz']; form.addEventListener('iron-form-submit', function() { alert('Form submitted with name: ' + form.serialize().name); return false; });

Filtering

Typing text into the vaadin-combo-box's input field will filter values defined in the items property and show only items that contain the value in the input field.

var combobox = combobox || document.querySelector("vaadin-combo-box"); combobox.items = ['foo', 'bar', 'baz'];

Sizing

In desktop browsers, the dropdown adapts it's size so as it does not overflow window edges and does not exceed input width. It also changes it's position from bottom to top if there is not enough room below the input.
Try to resize the window to check how the dropdown adapts its size and position.
